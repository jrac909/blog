读取模板 html 文件内容

先讲一下 function createRenderer 首先这个方法理解为一个工厂函数，vue ssr 中 createRenderer 和 createBundleRenderer 都是用来创建 render 的，render 是什么，就是将 vue 实例渲染成 html 字符串。因为有多个地方要调用 createBundleRenderer 方法，并且大部分参数是复杂但相同的，所以我们把它抽取出来，处理成一个方法， 就是这里的 createRenderer。

createBundleRenderer 的第一个参数是 bundle：这个其实就是以 entry-server.js 为入口打包生成的 bundle。（去看 ssr 教程就知道需要两个入口文件） 第二个参数是一个配置对象：这里其实就是指定了一些配置，可以看到，方法中的配置优先级更高，传入的参数只用作补充。

function createRenderer(bundle, options) {
  return createBundleRenderer(
    bundle,
    _.assign(options, {
      template, // 模板文件
      // 缓存配置
      cache: LRU({
        max: 1000,
        maxAge: 1000 * 60 * 15,
      }),
      // this is only needed when vue-server-renderer is npm-linked
      basedir: commonUtils.resolve('./dist'),
      // recommended for performance
      runInNewContext: false,
    }),
  );
}
